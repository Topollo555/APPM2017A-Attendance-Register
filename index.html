About APPM2017A Attendance Register

Features

  • ✓ Geolocation-based attendance verification
  • ✓ Local data storage with cloud backup
  • ✓ Admin dashboard with analytics
  • ✓ Calendar view for attendance tracking
  • ✓ CSV export functionality

How It Works

Students enter their details and check in. The system verifies their location against the lecture venue coordinates and records their attendance status.

Technical Details

This application uses HTML5 Geolocation API and stores records in browser localStorage with cloud synchronization via Google Sheets.
